### Description
**Stopwatch** app written in Javascript. Displays time that passed from pressing Start button in hours, minutes, seconds, and centiseconds. All data is stored in browser's local storage, can be accessed after closing the app.

### Features:
- accurate: using *JS Date()* object to measure passed time
- stores data in local storage, works after reloading the page
- displays laps, with highlighting of the fastest and slowest ones
