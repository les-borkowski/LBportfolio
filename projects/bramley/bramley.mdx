### Description
Python App takes url as in input, extracts text from given url, then counts occurrences of words in that text.
Results are served in json format by REST API and stored in a database.

### Features:
- Python app with unit tests
- REST API, with user authentication and local database to store users data
- health endpoint to check app and database status
- containerized using Docker-compose
