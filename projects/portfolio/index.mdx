### Description
**Portfolio** website made with React, using GatsbyJs, Sass and GraphQL. Projects are imported from markdown files and generated automatically.

### Features:
- content imported from markdown files through Gatsby's data layer
- styling done with Sass, organised by modules
- responsive desing using plain CSS
